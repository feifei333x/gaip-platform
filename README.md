# 生成式人工智能赋能商业数据分析与管理决策虚拟仿真实训平台（正式风格升级版）

## 文件说明
- `index.html`：平台主页面
- `style.css`：界面样式（已升级为正式实训平台风格）
- `app.js`：交互逻辑、角色登录、任务管理、数据分析与报告生成
- `data/`：3 套内置真实教学数据 CSV
- `assets/logo.svg`：平台标识

## 主要升级点
1. 界面由“演示页”升级为“正式平台式布局”
2. 新增左侧导航、角色页面、统计卡片、图表区
3. 新增 3 套真实教学数据：销售、客户行为、库存决策
4. 支持内置数据集加载、CSV 上传、数据预览、图表分析、AI 提示、自动报告
5. 教师端支持任务管理、提交评分，管理员端支持平台概况与日志展示

## 部署方法
将全部文件上传到 GitHub 仓库根目录后，提交并等待 GitHub Pages 自动更新即可。
